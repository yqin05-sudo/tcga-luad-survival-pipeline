# Clinical Variable Notes

This document summarizes the meaning and sources of clinical variables used in this project.

## Core Variables
- **case_id** – TCGA case identifier  
- **gender**  
- **age_at_index** – age at diagnosis  
- **vital_status** – Alive/Dead  
- **days_to_birth / days_to_death**  
- **days_to_last_follow_up**

## Diagnosis Variables
- **ajax_pathologic_stage**  
- **ajcc_pathologic_t**  
- **ajcc_pathologic_n**  
- **ajcc_pathologic_m**  
- **primary_diagnosis**

## Exposures
- **tobacco_smoking_status**  
- **pack_years_smoked**

## Follow-up (Preoperative)
- **ecog_performance_status**
- **karnofsky_performance_status**

## Engineered Treatment Variables
- **has_surgery** – derived from treatment_type  
- **has_chemotherapy**  
- **has_radiotherapy**
